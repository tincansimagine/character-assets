 v1.2.3 업데이트 (신규)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 버그 수정
  - 실리태번 업데이트 후 이미지/ZIP 업로드가 실패하던 문제 수정
  - 업로드/삭제 실패 시 원인을 알 수 있는 상세 에러 메시지 표시
    (CSRF 만료 시 "페이지 새로고침" 안내 등)
  - 갤러리/원본 미리보기가 모바일에서 화면 밖(페이지 맨 위)에
    떠서 닫을 수 없던 문제 수정 (브라우저 top-layer 방식으로 변경)
  - 삭제 확인 팝업의 버튼 글자가 세로로 쪼개지던 문제 수정
  - 반투명 테마에서 탭-패널 경계선이 비쳐 보이던 문제 수정

■ 신규 기능
  - 태그 복사: 에셋 카드의 태그(🏷) 아이콘으로 {{img::파일명.ext}}
    태그를 바로 클립보드에 복사
  - 개별 이름 변경: 연필(✏) 아이콘으로 에셋 하나씩 파일명 변경
  - WebP 자동 변환: 업로드 시 PNG/JPEG를 WebP로 자동 변환해
    저장 용량 절약 (에셋 탭 체크박스, GIF 제외)
  - 덮어쓰기 확인: 같은 이름의 파일 업로드 시 덮어쓸지 미리 확인
  - 설정 백업/복원: 도구 탭에서 확장 전체 설정(프리셋, 캐릭터별
    설정, 렌더러 옵션)을 JSON으로 내보내고 복원

■ UI 개선
  - 확장 헤더에 버전 배지 표시
  - 에셋 카드 액션 아이콘 터치 영역 확대 및 호버 피드백 개선
  - 노트 탭 스타일: 활성 탭이 아래 패널과 이어져 보이도록 개선
  - 버튼 아이콘-글자 간격, 버튼끼리 간격 전반 확대
  - 테마 호환성 개선: 밝은 테마 등 어떤 테마에서도 카드 경계선과
    안내 문구({{...}} 표기 등)가 항상 보이도록 색상 계산 방식 변경

SillyTavern에서 캐릭터 이미지 에셋을 관리하고, AI가 대화 맥락에 맞는
이미지를 자동으로 삽입하도록 돕는 확장입니다.

 UI 구조
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

확장 패널은 4개의 탭으로 구성되어 있습니다.

┌─────────────────────────────────────────────────────┐
│  프롬프트  │  에셋  │  렌더링  │  도구              │
└─────────────────────────────────────────────────────┘

상단 상태바에서 현재 캐릭터의 에셋 현황을 확인할 수 있으며,
[새로고침] / [프롬프트 복사] 버튼으로 빠르게 작업합니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 1. 프롬프트 탭
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AI에게 이미지 삽입 방법을 안내하는 프롬프트를 관리합니다.

■ 프리셋 시스템
  - 기본 프리셋 (범용), 1:1 감정/표정형, 시뮬레이션/장소·아이템형이
    기본 제공됩니다.
  - [새 프리셋] 생성, [이름 변경], [삭제], [저장]이 가능합니다.
  - JSON으로 프리셋을 [내보내기] / [불러오기]할 수 있어 다른 사용자와
    프리셋을 공유할 수 있습니다.

■ 지원 매크로 (프롬프트 안에 넣으면 자동 치환)
  {{img_inprompt}}          — 현재 설정된 이미지 출력 프롬프트 전체
  {{img_keywords_autogen}}  — 자동 감지된 에셋 파일명 목록 (확장자 포함)
  {{img_keywords_grouped}}  — 그룹핑된 키워드 목록
                              예: angry_[1, 2, 3], happy, sad
  {{img_keywords}}          — 수동 설정 키워드 목록
  {{charkey}}               — 현재 캐릭터의 실제 폴더명
  {{img_rand::PREFIX}}      — 프리픽스 기반 랜덤 에셋 URL
                              예: {{img_rand::angry}} → angry 그룹에서 랜덤
  {{img_count}}             — 현재 활성 에셋 총 개수
  {{img_resolve::NAME}}     — 에셋 이름으로 URL 직접 반환

■ [키워드 교체] 버튼
  프롬프트 본문 내 {{img_keywords}} 플레이스홀더를 현재 파일명 목록으로
  실제 치환합니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 2. 에셋 탭
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

캐릭터별 이미지 에셋을 업로드·관리합니다.

■ 에셋 링크
  - 다른 캐릭터의 에셋을 공유할 수 있습니다.
  - 예: 같은 캐릭터의 한국어/영어 봇카드가 따로 있을 때, 한쪽에만
    에셋을 넣고 나머지는 링크로 연결하면 용량을 절약합니다.
  - 링크된 상태에서는 업로드/삭제가 비활성화됩니다.

■ 에셋 버전 (프리셋)
  - 한 캐릭터의 에셋을 여러 버전으로 나눠 관리합니다.
  - 예: "헤테로", "BL", "공통" 등 상황별 에셋을 분리해두고 필요할 때
    버전만 전환하면 됩니다.
  - 매번 삭제→재업로드 없이 [추가], [이름 변경], [삭제]로 관리합니다.

■ 에셋 업로드
  - [업로드] — PNG, WebP, GIF, JPEG 이미지 파일 다중 선택 업로드
  - [ZIP] — ZIP 아카이브 통째로 업로드 (자동으로 압축 해제 후 등록)
  - 드래그 앤 드롭 — 에셋 목록 영역에 이미지나 ZIP을 끌어다 놓으면
    자동 업로드됩니다.

■ 에셋 검색 & 필터
  - 검색창에서 키워드나 파일명으로 실시간 필터링합니다.

■ 에셋 On/Off (개별 활성/비활성)
  - 에셋 목록에서 눈(👁) 아이콘을 눌러 개별 에셋을 비활성화할 수
    있습니다.
  - 비활성화된 에셋은 AI 프롬프트와 렌더러에서 제외되지만, 파일은
    삭제되지 않습니다.
  - 다인봇에서 특정 캐릭터의 에셋만 활성화하거나, 토큰 절약을 위해
    일시적으로 에셋을 끌 때 유용합니다.

■ 일괄 선택 & 삭제
  - [전체 선택] 체크박스로 모든 에셋을 선택한 뒤 [선택 삭제]로 한번에
    삭제할 수 있습니다.

■ 갤러리
  - [갤러리] 버튼으로 이미지 썸네일을 그리드 형태로 확인합니다.
  - 그리드 크기 조절: S(소) / M(중) / L(대)
  - 갤러리 내 검색으로 특정 에셋을 빠르게 찾습니다.
  - 이미지 클릭 시 라이트박스(원본 크기 미리보기)가 열립니다.
    ← → 화살표 버튼 또는 키보드 좌/우 방향키로 이전/다음 이미지를
    탐색할 수 있습니다.
  - 멀티셀렉트 모드: 체크(✓✓) 버튼을 눌러 활성화하면 여러 이미지를
    클릭하여 선택한 뒤, 휴지통 버튼으로 한꺼번에 삭제합니다.

■ 이름 정돈 (Smart Rename)
  - [정돈] 버튼으로 모든 에셋 파일명을 일괄 변환합니다.
  - 옵션:
    · 공백 → 언더바(_)
    · 특수문자 제거 (알파벳, 숫자, 한글, _, - 외 제거)
    · 언더바 정리 (연속/앞뒤 언더바 제거)
    · 소문자로 변환
    · 점(.) → 언더바(_) (name.1 → name_1)
    · 이름숫자 → 이름_숫자 (angry1 → angry_1)

 3. 렌더링 탭
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AI가 출력한 {{img::파일명.ext}} 태그를 실제 이미지로 변환합니다.

■ 내장 렌더러
  - Regex 확장 없이도 바로 이미지를 표시할 수 있습니다.
  - 기존 Regex 방식과 공존 가능 (단, 동시 사용 시 이미지 중복 주의).

■ 랜덤 에셋 출력
  - 같은 키워드 그룹(예: angry_1, angry_2, angry_3)이 있으면 AI에게는
    키워드(angry)만 전달됩니다.
  - 렌더링 시 해당 그룹 내에서 랜덤으로 하나를 표시합니다.
  - 파일명 구분자로 _  -  . 모두 지원합니다.
    (angry_1, angry-2, angry.3 전부 인식)

■ 이미지 지연 로딩 (Lazy Loading)
  - IntersectionObserver 기반으로 화면에 보이는 메시지만 렌더링합니다.
  - requestIdleCallback으로 렌더 패스를 스케줄링하여 스크롤 버벅임을
    최소화합니다.
  - CSS containment(contain: content)로 긴 채팅의 리플로우를 줄입니다.
  - 대량 이미지가 있는 긴 채팅에서 성능을 크게 개선합니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 4. 도구 탭
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 페르소나 에셋
  - 유저(페르소나)의 공통 에셋을 업로드합니다.
  - 캐릭터 에셋에서 키워드를 찾지 못하면 페르소나 에셋에서 자동으로
    폴백 검색합니다.
  - 모든 캐릭터 채팅에 공통으로 사용할 이미지(배경, UI 요소 등)를
    한 곳에 모아둘 수 있습니다.
  - 이미지 업로드, ZIP 업로드를 지원합니다.

■ 백업 & 내보내기
  - [에셋 ZIP 내보내기] — 현재 캐릭터의 모든 에셋을 ZIP 파일로
    다운로드합니다. (JSZip CDN 사용)
  - [프롬프트 복사] — 현재 프롬프트를 클립보드에 복사합니다.

■ 사용 가이드
  - 기본 사용법을 접었다 펼쳐서 빠르게 확인합니다.

■ Regex 스타일 다운로드
  - 기존 Regex 확장을 사용하는 분들을 위해 미리 만들어진 스타일
    JSON을 다운로드할 수 있습니다.
  - 스타일 종류 (9가지):
    · 기본형 (둥근 사각형)
    · 테두리형 (테두리 + 그림자)
    · 깔끔형 (투명 배경)
    · 그림자형 (강한 그림자)
    · 폴라로이드 (사진 프레임)
    · 시네마틱 (레터박스)
    · 왼쪽 플로팅 (텍스트 감싸기)
    · 오른쪽 플로팅 (텍스트 감싸기)
    · 비네팅 (어두운 테두리)
  - [모든 스타일 다운로드] 버튼으로 전체를 한꺼번에 받을 수 있습니다.

 5. 그룹 채팅 지원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 그룹 채팅(다인봇) 진입 시 모든 멤버의 에셋을 자동으로 로드합니다.
- 키워드 목록과 렌더러 캐시에 전체 멤버의 에셋이 반영됩니다.
- 에셋 탭에서 멤버별 에셋 요약을 확인할 수 있습니다.
- 개별 캐릭터의 에셋 On/Off가 그룹에서도 적용됩니다.

 6. 슬래시 명령어
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

채팅창에 입력하여 사용할 수 있는 명령어입니다.

/assets-upload label="키워드" URL
  — URL에서 이미지를 받아 에셋으로 업로드합니다.

1) 캐릭터를 선택합니다.
2) [에셋] 탭에서 이미지를 업로드하거나 ZIP을 올립니다.
   (드래그 앤 드롭도 가능)
3) [렌더링] 탭에서 "내장 렌더러 사용"을 켭니다.
   (또는 [도구] 탭에서 Regex 스타일 JSON을 다운로드하여 Regex 확장에
    등록합니다.)
4) [프롬프트] 탭에서 프리셋을 선택하고, 캐릭터의 시스템 프롬프트 또는
   Author's Note에 {{img_inprompt}} 매크로를 삽입합니다.
5) AI가 {{img::파일명.ext}} 형식으로 출력하면 이미지가 자동으로
   표시됩니다.
